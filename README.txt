Dit project voor Visual Studio 2010 bevat de voorbeeldcode die hoort bij de 
tutorial op http://www.youtube.com/watch?v=aTMHlit1Ggw

Pas de connection strings aan in Backend.Test/App.Config en Service/Web.Config
om deze demo te gebruiken.

In Backend/Data.sql vind je de definitie van de stored procedure die door de
backend wordt gebruikt.
